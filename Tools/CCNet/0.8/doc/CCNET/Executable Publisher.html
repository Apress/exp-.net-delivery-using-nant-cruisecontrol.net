Executable Publisher

The executable publisher simply starts an executable and waits for it to exit. You may access the last build label via the ccnet.label environment variable.

Typical Configuration

<publisher type="executable"
           executable="somefile.exe"
           arguments="arg1 arg2"
           workingDirectory="C:\SOME_DIRECTORY" />

Full Configuration

<publisher type="executable"
           executable="somefile.exe"
           arguments="arg1 arg2"
           timeout="120000"
           workingDirectory="C:\SOME_DIRECTORY"
           nonzeroExitFatal="true" />

Element Description Type Required Default
executable The file that will be executed. string yes none
arguments The arguments that are passed to the executable. string no no arguments
timeout The time that the process is allowed to execute for before it is killed. Measured in milliseconds. int no 60 seconds
workingDirectory The directory in which the executable will run. string no CCNet's current directory
nonzeroExitFatal Whether a nonzero exit status will throw an exception. bool no false
