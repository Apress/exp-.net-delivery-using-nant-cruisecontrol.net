We use our Sourceforge project to host public mailing lists. To subscribe to lists, or read their archives, click here.

We have 3 primary mailing lists:

ccnet-user - A general purpose mailing lists for support, announcements, usage discussions, etc. Use ccnet-user if you need help using CruiseControl.NET

ccnet-devel - This is where we discuss all things to do with the continuing development of CCNet

ccnet-checkins - All CVS commits to the CCNet project tree are logged here.

GMane Archives

CCNet mailing lists are also archived at GMane (although the archives don't go back to the beginning of the lists). The relevant URLs are as follows:
